infix_forward_t* infix_reverse_t::cached_forward_trampoline

A pre-compiled forward trampoline for calling the user's C callback.

This is the core of the reverse call mechanism's type safety and ergonomics. The C dispatcher uses this trampoline to correctly marshal the void** arguments and the context pointer into the native ABI required to call the user's clean, type-safe C handler function.
