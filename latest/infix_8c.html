Detailed Description

The unity build source file for the infix library.

Copyright
Copyright (c) 2025 Sanko Robinson

This file is the single translation unit for the entire infix library. It includes all other necessary C source files in the correct order to create the final library object. This approach simplifies the build process and can enable more aggressive compiler optimizations.

All functions in the included files (except for the public API declared in infix.h) should be declared as static to ensure they are properly encapsulated within this translation unit.

Note
This file is not intended to be compiled on its own without the rest of the source tree. It is the entry point for the build system.
