Detailed Description

Tests the high-level string-based signature API.

This test suite verifies the functionality and robustness of the infix_forward_create API. It covers:

  1. Happy Paths: Correctly parsing and generating trampolines for simple primitive, pointer, and variadic function signatures.
  2. Advanced Packed Structs: Verifying the parsing of the complex p(size,align){type@offset...} syntax.
  3. Detailed Function Pointers: Verifying that function pointer types are parsed recursively into detailed INFIX_TYPE_REVERSE_TRAMPOLINE types.
  4. Error Handling: Ensuring the parser correctly rejects a wide variety of malformed and invalid signature strings.
