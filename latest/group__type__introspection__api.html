Detailed Description

Functions for safely querying the properties of infix_type objects.

These read-only functions provide a safe and convenient way to inspect the metadata of a parsed infix_type without needing to directly access the members of the infix_type_t struct. They include null-safety checks and return sensible default values for invalid or mismatched types.
