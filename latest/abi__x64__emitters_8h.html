Detailed Description

Declares the internal helper functions for emitting x86-64 machine code.

This is a non-public, internal-only header. It provides the function prototypes for all low-level x86-64 instruction emitters.

This module was created to cleanly separate platform-specific implementation details from the generic trampoline engine. These functions are shared by both the Windows x64 (abi_win_x64.c) and System V x64 (abi_sysv_x64.c) ABI implementations, providing a consistent, low-level interface for generating machine code for the x86-64 architecture. Each function corresponds to a specific machine instruction or a common addressing mode.

Emits mov r64, imm64 to load a 64-bit immediate value into a register.

Emits mov r64, imm64 to load a 64-bit immediate value into a register.

This is the most direct way to load a full 64-bit immediate value (like a pointer or a large constant) into a general-purpose register.

  • Instruction format: REX.W + B8+r imm64
  • REX.W: A mandatory prefix (0x48) that promotes the operation to 64 bits.
  • B8+r: The base opcode for moving an immediate into a register, where r is the low 3 bits of the register's index.
  • imm64: The 8-byte immediate value.
Parameters
bufThe code buffer to write to.
regThe destination 64-bit register (GPR).
immThe 64-bit immediate value to load.

Emits pop r64 (pops a 64-bit value from the stack into a register).

Pops a 64-bit value from the top of the stack into the specified register. This also increments the stack pointer (RSP) by 8.

  • Opcode: 58+r (where r is the register index).
  • A REX prefix is added if the register is R8 or higher.

Emits an x86-64 REX prefix byte, used to enable 64-bit operations and extended registers.

Emits an x86-64 REX prefix byte, used to enable 64-bit operations and extended registers.

The REX prefix is a single byte (in the range 0x40-0x4F) used in 64-bit mode to enable features not available in 32-bit mode.

Parameters
bufThe code buffer.
wIf true, sets the W bit (bit 3), promoting the operand size to 64 bits.
rIf true, sets the R bit (bit 2), extending the ModR/M reg field to access registers R8-R15.
xIf true, sets the X bit (bit 1), extending the SIB index field.
bIf true, sets the B bit (bit 0), extending the ModR/M r/m field or SIB base field.
