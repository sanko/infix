Detailed Description

A dual-purpose fuzzing harness for libFuzzer (Clang) and AFL++ (GCC), focused on the infix FFI trampoline generation API.

This harness uses the shared recursive generator (fuzz_helpers.h) to create a pool of complex infix_type objects within a memory arena. It then uses these types to construct randomized function signatures which are passed to infix_forward_create_manual and infix_reverse_create_manual.

The goal is to find bugs in the ABI classification and JIT code generation stages. This harness now tests the fully arena-based workflow.
