typedef void(* infix_cif_func) (void *target_function, void *return_value, void **args)

The signature for a generic forward-call trampoline, the "Call InterFace" function.

This is the function pointer type returned by infix_forward_get_code. It provides a standardized way to invoke any C function for which a trampoline was generated.

Parameters
target_functionA pointer to the native C function to be called.
return_valueA pointer to a buffer where the return value will be stored.
argsAn array of pointers, where each element points to an argument's value.

Creates a new infix_type for a packed struct from an arena.

This is the designated way to describe C structs that use non-standard memory layouts, typically resulting from compiler directives like __attribute__((packed)) or #pragma pack(1). All memory is allocated from the provided arena.

Parameters
arenaThe memory arena from which to allocate the new infix_type.
[out]out_typeOn success, this will point to a newly created infix_type.
total_sizeThe exact size of the packed struct in bytes, as returned by sizeof(your_packed_struct).
alignmentThe alignment requirement of the packed struct in bytes, as returned by _Alignof(your_packed_struct). This is often 1.
membersAn array of infix_struct_member describing each member of the struct. The offsets within this array must be the correct, packed offsets from offsetof.
num_membersThe number of elements in the members array.
Returns
INFIX_SUCCESS on successful creation.
INFIX_ERROR_INVALID_ARGUMENT if out_type is nullptr, alignment is zero, or members is nullptr when num_members > 0.
INFIX_ERROR_ALLOCATION_FAILED if memory for the infix_type struct could not be allocated from the arena.
Note
Memory Ownership: All memory for the created infix_type and its metadata is owned by the provided arena.

◆ infix_type_create_primitive()

c23_nodiscard infix_type * infix_type_create_primitive ( infix_primitive_type_id  id)

Creates an infix_type descriptor for a primitive C type.

This function returns a pointer to a static, singleton instance for the requested primitive type. These do not need to be freed.

Parameters
idThe enumerator for the desired primitive type (e.g., INFIX_PRIMITIVE_SINT32).
Returns
A pointer to the static infix_type descriptor. Returns nullptr for invalid IDs.
Warning
The returned pointer must NOT be passed to any deallocation function.

This function acts as a factory, returning a pointer to one of the statically allocated, singleton infix_type instances for primitives. It correctly handles platform-specific type definitions, such as long double being equivalent to double on MSVC and Clang for Windows, ensuring the correct type descriptor is used for the target ABI.

Parameters
idThe ID of the primitive type (e.g., INFIX_PRIMITIVE_SINT32).
Returns
A pointer to the corresponding static infix_type structure. Returns nullptr if the ID is invalid or for a type not supported by the current compiler (e.g., __int128_t on MSVC).
Warning
Do not free the returned pointer. It points to a static global variable.

Creates a new infix_type for a struct from an arena.

This function calculates the size and alignment of the struct based on its members, adhering to standard C layout rules. It iterates through the members to find the maximum alignment requirement and calculates the total size, including any trailing padding needed to satisfy the struct's overall alignment. It is hardened against integer overflows that could arise from malformed member layouts.

membersAn array of infix_struct_member describing the struct's layout.

INFIX_SUCCESS on success, or an error code on failure.
